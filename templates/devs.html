for developers

everything on this explorer is open, scriptable, and attributable. this page is the hub for devs integrating the data or the broader attestation and custody stack it comes from.

no auth, no api key, no rate limit. all ssr, curl-friendly.

observability kit

pre-configured grafana dashboard at ctaz-crosslink-grafana.json. import into grafana and point at a prometheus scraping /metrics. scrape config and alert rule suggestions on the metrics guide.

open source

explorer source at Frontier-Compute/ctaz-explorer, mit license. clone it, point at any crosslink node rpc, run the same pages. no proprietary deps, fastapi and jinja.

if you are building on zcash attestation

the /anchors view and /api/verify/<txid> endpoint on this explorer are powered by zap1, an open attestation protocol anchored on zcash mainnet with ~3,000 npm installs per month across its client packages. full api at api.frontiercompute.cash/docs, spec at openapi/zap1.yaml. deployed on eth l1, arbitrum, base, hyperliquid, near, sui.

if you are running agents that hold capital

shieldedvault is a custody standard for agents where the agent cannot steal, cannot lie, cannot survive its owner. split-key mpc, zcash mainnet attestation, open spec. free tier available. npx create-shieldedvault my-agent to start.
