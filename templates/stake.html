support this explorer

this explorer is free and open source, runs on a single vps behind a cloudflare tunnel, maintained by zk-nd3r. there are no ads, no analytics, no api keys, no rate limits.

zooko suggested in the crosslink s1 workshop signal group that explorer operators should run a finalizer and accept delegated stake as a funding model. that is the plan for this one.

how to support this operator

copy the finalizer pub key below and, in the crosslink desktop wallet during a staking day window, paste it as the target and stake any amount. delegated stake on the active roster earns block rewards, those rewards fund the server, and at the end of season 1 the real zec payout is distributed pro rata to miners and stakers based on block rewards earned.

what this operator promises

  • the explorer stays up, or a transparent status is shown when it is not

  • no bond retargeting to anyone other than this same pub key without a visible notice on this page
  • source code stays mit licensed on the github repo linked in the footer
