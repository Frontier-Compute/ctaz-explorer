what this is

a single-block canonical hash lookup against our node. useful when your local node has been mining in isolation and you want to know before you spend more time on a sidechain. hash comparison is case-insensitive, expects lowercase hex, 64 characters.

json api

programmatic: /api/sync-check/<height>/<your_hash>. returns {height, canonical_hash, your_hash, match, our_tip, our_peers}. curl-friendly, no auth, no rate limit.

context

multiple nodes in the crosslink season one feature net forked their own sidechains starting apr 17 following an upstream server node that fell behind after bft block 1337. if you have been stuck at the same height for an unusually long time or your miner keeps winning every block, check here first. recovery snapshots at community snapshot endpoint or in the crosslink updates channel.
